# Optional pi-spec Subagents

These files are optional project-level agent definitions for users of `pi-subagents`.

Install them into a project with:

```text
/spec-install-agents
```

The command copies these markdown files into `.pi/agents/` so `pi-subagents` can discover them.

Included agents:

- `spec-brainstorm-agent`
- `spec-requirements-agent`
- `spec-design-agent`
- `spec-task-planner-agent`
- `spec-review-agent`

Use them for advisory drafting and review. Keep one primary writer responsible for applying final spec or code edits.
